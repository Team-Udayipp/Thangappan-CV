# Thangappan's Career Consulting 🧔

> *Professional CVs. Personal Touch. No Refunds.*

**Thangappan's Career Consulting** is a sophisticated, highly-interactive web prank disguised as a premium, corporate-grade CV builder service. It was designed specifically for the **IEEE Prankraft** frontend competition. 

What starts as a sleek, professional web application slowly devolves into a 20-stage psychological journey involving deep-seated personal questions, a deceptive Google Meet interview simulation, fake debit deductions, and a chaotic "Life Audit" — all guided by the eccentric, unyielding personality of Thangappan.

✨ **Live Demo:** [Click here to begin your consultation](https://thangappan-cv.vercel.app/)

---

## 🎭 The Experience

The application leverages a "Continuous Illusion" web architecture. As users click through perfectly stylized UI components (glassmorphism, tailored typography, micro-animations), the state-driven application responds with synchronized dialogue bubbles injected automatically. 

Key prank features include:
- **Responsive "High-End" UI:** Purposely designed to foster trust and legitimacy before pulling the rug out.
- **Dynamic Avatar System:** Thangappan’s face tracks progress, shaking when angry, swaying when thinking, and responding to direct clicks.
- **Fake Video Interview:** A mobile-responsive 2x2 grid simulating Google Meet, complete with a terrifying, muffled HR recruiter.
- **Psychological Escalation:** Slowly shifts from standard CV forms (Name, Skills) to intrusive queries (Relationship Status, Bank Balances).
- **Cinematic Pacing:** Features custom Web Audio API sound effects, page transition animations, and a CSS-rendered text corruption/glitch effect.

## 🛠️ Technology Stack

No frameworks, no bloated libraries. Built purely for performance and front-end mastery:
- **HTML5** & **Semantic Structuring**
- **Vanilla JavaScript (ES6)** (State management, audio orchestration, DOM manipulation)
- **Vanilla Custom CSS3** (CSS Variables, Flexbox/Grid, Keyframe Animations, Mobile-first media queries)

## 🚀 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/AlanAli-byte/Thangappan-CV.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Thangappan-CV
   ```
3. Because of the use of the Web Audio Context and local assets, you must run it through a local web server (opening the file directly via `file:///` may cause browser CORS errors with audio files). 
   - If using Node.js: `npx serve` or `node server.js`
   - If using Python: `python -m http.server 3333`
   - Or simply use the VS Code **Live Server** extension.
4. Navigate to `http://localhost:3333`.
